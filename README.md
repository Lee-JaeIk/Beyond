# Beyond

> 세종대학교 2016년 2학기 Capsone Design 설계 강의에서 진행한 팀프로젝트이며, 안드로이드(4.4, KitKat)과 Cyperss사 BLE 하드웨어가 결합된 IoT 프로젝트입니다.



### 개요
* Beyond는 네비게이션의 음성안내를 촉각과 시각으로 변환해주는 청각장애인 네비게이션 솔루션입니다. 안드로이드 애플리케이션과 웨어러블 디바이스로 이루어져 있습니다.
